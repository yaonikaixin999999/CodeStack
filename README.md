# CodeStack
本项目实现了一个名为 CloudStack-Pro 的云原生在线协同开发平台。
该平台旨在为开发者提供一个无需本地配置、随时随地可访问且支持多人实时协作的云端开发环境，极大地提升了软件开发效率和团队协作体验。 本系统采用前后端分离的微服务架构，前端基于Vue.js 3和Monaco Editor
构建，提供功能丰富、响应迅速的Web用户界面；后端采用Node.js+Express处理文件系统操作和实时通信，Spring Boot处理用户认证和数据持久化。系统核心创新在于双通道协同编辑模型：利用Y.js处理文档内容的无冲突合并，独立Socket.IO 通道传输光标位置、用户状态等元数据。此外，系统通过SSH协议与远程Linux 服务器深度集成，实现了远程文件管理、在线终端、编译运行和 AI辅助编程等功能模块。
